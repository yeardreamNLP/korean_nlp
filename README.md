# korean_nlp

## Description
- `data`: 학습데이터/평가데이터 csv 파일이 담긴 폴더.
- `inference.py`: LLM 모델 추론하는 python script
- `vllm_inference.py`: LLM 모델 추론을 vllm 통해 실험
- `inference.sh`: inference.sh 추론 쉘 실행 스크립트
